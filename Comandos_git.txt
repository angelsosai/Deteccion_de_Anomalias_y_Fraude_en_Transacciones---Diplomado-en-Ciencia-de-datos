
Primer uso del repositorio
1. Clonar el repositorio
git clone https://github.com/angelsosai/Deteccion_de_Anomalias_y_Fraude_en_Transacciones---Diplomado-en-Ciencia-de-datos.git
cd Deteccion_de_Anomalias_y_Fraude_en_Transacciones---Diplomado-en-Ciencia-de-datos

2. Crear tu rama de trabajo (solo una vez)

git checkout -b TU_RAMA

#CAMBIA DE RAMA

git switch TU_RAMA

3. Subir la rama al repositorio remoto (solo una vez)
git push -u origin TU_RAMA

4. Si Git no reconoce tu usuario

git config --global user.name "Angel Sosa"
git config --global user.email "angelsosa@ciencias.unam.mx"




Actualizar cambios del repositorio
git pull



CADA QUE NECESITES HACER CAMBIOS
#GUARDAR CAMBIOS
git add .

#Agrega descripcion
git commit -m "importar_data"

git push


#IMPORTAR LAS LIBRERIAS
pip install -r requirements.txt
